This table allows to manipulate skeletons in batches. Skeletons can be collected to then use in other widgets such as the 3D Viewer. Skeletons can be selected from skeleton sources such as Active skeleton or from other open widgets using the ‘From’ drop-down and the Append button.

Buttons

Clear
Removes ALL skeletons from the table
Refresh
Refreshes the table to reflect any updates made to listed skeletons such as Review status
Open
Skeleton batches previously saved can be loaded into the Selection Table
Save
Listed skeletons can be saved to a single json file
Annotate
Add annotations to selected skeletons
Color scheme
Specific color schemes can be selected to apply to the skeleton collection
Colorize
Applies selected color scheme to the skeleton collection
Measure
Opens the Measurement Table displaying basic metrics of selected skeletons
Summary Info
Opens a window displaying tracing metrics of selected skeletons
Append with batch color
Adds skeletons to the Selection table with current Batch color
Show visibility controls
Displays options to control visibility of skeletons in the 3D Viewer

Columns

name
Names of skeletons
Filter
Listed skeletons can be filtered by name and annotations
Rev
Review status of listed skeletons. Default is Union (total) Review percentage.
Visibility Controls
Show/hide pre-, post-synaptic links to/from connector node
Text
Displays labels in 3D viewer
Meta
Displays tags as orange spheres
Action: Annotation
Add annotation to a given skeleton
Action: Summary info
Displays tracing metrics of given skeleton
Action: Neuron Navigator
Open Neuron Navigator page of given skeleton
