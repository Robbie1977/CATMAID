Export Graphs for layouting in other tools

For some graphs, layouting is easier done in other tools, e.g. Gephi. Since the coloring is likely easier to do in CATMAID, a re-import is useful as well. This workflow can be done in CATMAID like the following:

  1. Export a graph as GML, the default export options (linearize IDs) are fine.
  2. Open GML file in Gephi and do layout.
  3. Export graph in Gephi as GraphML file: File > Export > Graph file. Choose "GraphML" as file type in file dialog.
  4. In CATMAID, click "Openâ€¦" in the Graph Widget\'s main tab and select the exported GraphML (ends on .graphml).
  5. The default options in the import dialog should work for Gephi. Preferring the unit_id and name fields assumes a previous CATMAID export of the graph.
